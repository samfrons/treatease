1. Getting Started

A few words on getting started with your newest Bootstrap project, package files & theme support.

1.1 Included Files

After downloading the theme package open the folder named “DASHIO - FrontEnd (Version)” and un-zip the folder, if it is not already so. Inside the package you will find the following files:

  • Licensing
  • Theme
  • Documentation

1.2 Installation Options

To install your brand new Bootstrap theme (that you’ve already downloaded and un-packed) you have two primary options: FTP or Web Hosting Upload Tools. I strongly recommend to use the first option, because it's easier and more reliable.

1.3 FTP Upload

To manually upload your new Bootstrap theme, login with your credentials to your website and locate your desired folder in your host server. Upload the un-zipped theme folder into the: yourdomain.com/ or yourdomain.com/folder/

Please ensure you are uploading the un-zipped theme folder, not the entire package downloaded.

1.4 Theme Support

If you have a problem with the theme or found a bug please let me know by send me an email through carlos@alvarez.is and I will do what I can to assist you.

In order to submit me a Support Question, you will need to locate your Theme's Purchase Code, provided by the marketplace.

I tested the theme with major browsers and different devices. Unfortunately it is impossible to run every test scenario. I'll try to keep the theme as healthy as I can.

Note: I am happy to help with setup and any bugs you may stumble upon though I cannot support CSS modifications to your theme.

3. Theme Configuration

A quick guide to help you with the basic styles of your theme.

3.1 Google Fonts

This theme uses a pre-defined Google Fonts, 'Ruda'. If you want to change this value, you should take two steps. Replace the "Link Href" reference on the HTMLs pages and second, replace all references of 'Ruda' with your new font.

3.2 Animation

The theme uses Jquery.Animate Enhanced script and pre-defined CSS (included in style.css file) to animate the objects.

If you need examples of usage or more information about the script, please visit This Page. Also, check the effect attribute in the style.css file.

3.3 Font Awesome 4.0.3

Font Awesome gives you scalable vector icons that can instantly be customized — size, color, drop shadow, and anything that can be done with the power of CSS.

The theme uses font awesome in different parts of the code. Each part has their own configuration and styles. Please, refer to the next page to see the full list of icons available in this version and examples of use. FontAwesome.io.

3.4 Theme Colors

This theme comes with a pre-set of colours configured. If you want to change the color of your theme, just modify the color reference in your html with your desired color file. By default, the themes uses the file brown.css, green.css and blue.css. If you want to use a specific color, just edit one of the .css files included in the color folder inside the CSS folder. Replace all references of one color with other.

3.5 Hover Image - Color Effect

Maxfront uses a set of javascripts to manage the hover effect when you put your mouse an image. If you want to change the color of the hover effect or add some other effects to the image, please refer to jquery.hoverdir.js file. This file was created by Codrops.com.

3.6 Layerslider Script

Maxfront uses Layerslider script (demo on index2.html & index3.html). All files are included in Layerslider folder. This script was created by Kreaturemedia.com.

3.7 Isotope Script

Some portfolio options & pages uses Isotope script to re-arrange the images in the site. To deep understanding of this script, please visit this site.

3.8 Easy Pie Chart Script

Charts are created using Easy Pie Chart Script. (demo in about.html). The main attributes of the chart is controlled by the jquery.easypiechart.js script. The color, size and other basic configurations are detailed on each html file.

More demos and information here: Site.

3.9 Retina Script

This theme was created with support to retina devices. Retina.js script is used to enhance the experience of retina display's users.

The JavaScript script automatically replaces images on your page with high-resolution variants (if they exist). Steps to use it:

  • 1 - Place the retina.js file on your server
  • 2 - Include the script on your page (script type="text/javascript" src="/scripts/retina.js" /script)
  • 3 - That's it!

More information here.
